# WeatherAI
使用火山引擎官方提供SDK，嵌入AI助理为你提供出行穿衣建议  
天气查询 APP，不仅能够精准地提供天气信息，还独具匠心地融入了 AI 助理
功能，与此同时，AI 助理基于天气数据，能为用户提供个性化的出行建议，无论
是选择出行方式、规划路线，还是提醒携带雨具等，都能给予贴心的指导。
